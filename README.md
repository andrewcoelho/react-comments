# React Comments
---

### Demo
[http://andrewcoelho.com/react-comments/](http://andrewcoelho.com/react-comments/)

### Installation
```bash
git clone https://github.com/andrewcoelho/react-comments.git
cd react-comments
npm install
```
### Developing
```bash
npm run dev
```

### Testing
```bash
npm run test
# or
npm run test:watch
```

### Production
```bash
npm run build
npm start
```
